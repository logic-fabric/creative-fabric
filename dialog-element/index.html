Dialog element implementation

Source: Codrops article How to Implement and Style the Dialog Element

Good practices for a dialog element

  • Focus the first interactive element in the modal once the dialog opens (native for HTML dialog element)
  • Trap focus inside the modal
  • Provide at least one button to close the dialog
  • Prevent interaction with the rest of the page (native for HTML dialog element)
  • Close the dialog when the user presses ESC (native for HTML dialog element) or disable it
  • Return focus to the element that opened the dialog in the first place
